# 💰 Finanças Pessoais

Plataforma pessoal de controle financeiro — hospedada via GitHub Pages.

## Funcionalidades

- 📅 Organização por mês com abas
- 💵 Registro de salários
- 🔒 Dívidas fixas mensais
- 💳 Cartões de crédito (dia 08, 15, 23 e 25) com parcelas automáticas
- 📒 Movimentos do dia (gastos avulsos e recebimentos extras)
- 📈 Investimentos com projeção 105% CDI
- 💾 Dados salvos automaticamente no navegador (localStorage)

## Como usar

Acesse sua URL do GitHub Pages e use direto no navegador.
Todos os dados ficam salvos **localmente no seu dispositivo** — nenhuma informação é enviada a servidores.
